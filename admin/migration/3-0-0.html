Version 3.0.0

The container was updated and requires slightly different parameters now.

With helfertoolctl

Make sure that you have helfertoolctl version 0.9.0 or later. It will take care of all required changes automatically.

You can remove the following configuration from helfertool.yaml:

docker: false

Other deployments

The docker run command with the correct parameters can be found in the helfertoolctl script.

An example for podman can be found here.
